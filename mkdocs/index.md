# RESTify Experiment Replication Package

Everything you ever need to validate our findings, in one place.

## About

As part of our submission to the [ESEM 2023 conference](https://conf.researchr.org/series/esem), we offer a replication package that allows fast and independent replication of all our results and interpretations.  
This page serves as starting point for the various components needed.

 > Note: This is an anonymized version of the replication package, created specifically for the reviewers. In case of an accept we will provide a link to the original replication package.  
We want to highlight, that the anonymization process affects the replicability, e.g. when java packages were renamed or parts of links were anonymized. Wherever the case, we add a marker "[A]" to indicate reduced functionality due to temporary anonymization.
